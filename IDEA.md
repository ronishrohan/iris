# iris 🌸

a voice assistant that literally replaces Siri on macOS.

always listening for a wake phrase ("hey iris" by default, configurable). once triggered, it records what you say, understands it, does the thing, and talks back.

the brain is an LLM — deepseek by default (BYOK), but swappable with a local model for fully offline use. it has access to real tools: run terminal commands, send messages, set reminders, control spotify, search the web. not just answering questions — actually doing things on your mac.

runs silently in the background. starts with your mac. feels like siri but actually works.
